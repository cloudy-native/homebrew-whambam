# homebrew-whambam

This repository contains the Homebrew formula for `whambam`.

## What is whambam?

`whambam` is a simple CLI tool for web performance testing.

## Installation

To install `whambam` with homebrew, run:

```bash
brew install cloudy-native/whambam/whambam
```
